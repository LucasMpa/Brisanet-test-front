## Como Iniciar Na Sua Máquina

Para instalar as dependências:

```
 $ npm i
 
 ou
 
 $ yarn
```

Para iniciar o projeto:
```
 $ npm start
 
 ou
 
 $ yarn start
```


Clique <a href="https://test-pokedex-front.netlify.app/">aqui</a> para acessar a aplicação funcionando.
